### Auto-Complete
- Type "/" in chat input
- Suggestions appear as you type
- Arrow keys to navigate
- Enter to insert
- Tab to preview

---

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `/` | Start shortcut in chat |
| `Cmd+K` | Open response picker |
| `Enter` | Insert selected |
| `Tab` | Preview response |
| `Esc` | Close picker |

---

## Behaviors

### Variable Replacement
- Variables replaced when response is sent
- Missing variables show placeholder
- Preview shows sample data

### Usage Tracking
- Track how often each response is used
- Sort by popularity
- Analytics for team responses

### Permissions
- Personal responses: Only creator
- Team responses: All agents can use, only creator/admin can edit
