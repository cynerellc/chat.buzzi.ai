## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line |
| `Cmd+K` | Canned responses |
| `Cmd+E` | Emoji picker |
| `Cmd+Shift+A` | Attach file |
| `Esc` | Back to inbox |
