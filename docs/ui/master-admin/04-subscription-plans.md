## Behaviors

### Plan Validation
- At least one plan must be active
- Cannot delete plan with active subscribers
- Price changes don't affect existing subscribers

### Subscriber Migration
- When archiving, suggest migration path
- Bulk migration tool for moving subscribers
- Prorated billing for plan changes

### Change Propagation
- Feature changes apply immediately to new conversations
- Limit changes apply to next billing cycle
- Price changes apply to new subscribers only
