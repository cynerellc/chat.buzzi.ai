## Behaviors

### Auto-Save vs Manual Save
- **Auto-save**: Toggle switches save immediately
- **Manual save**: Text fields require "Save Changes" button

### Validation
- Required fields validated before save
- Invalid values show inline errors
- Confirm dialogs for dangerous changes

### Change History
- All setting changes logged to audit
- "Last modified by" shown per section
- Ability to view previous values

---

## Mobile Layout

- Settings menu as collapsible accordion
- Full-width form fields
- Floating save button
- Sensitive fields with full-screen modal edit
