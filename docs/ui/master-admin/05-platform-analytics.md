## Chart Interactions

### Common Controls
- **Hover**: Show tooltip with exact values
- **Click Legend**: Toggle series visibility
- **Zoom**: Drag to select range
- **Reset**: Double-click to reset zoom

### Chart Types Used
| Chart | Purpose |
|-------|---------|
| Line | Trends over time |
| Area (stacked) | Composition over time |
| Bar | Comparisons |
| Pie/Donut | Distribution |
| Heatmap | Time patterns |
| Histogram | Distribution |
