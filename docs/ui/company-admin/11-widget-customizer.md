- Updates in real-time as you change settings
- Can interact with preview (click launcher, test chat)
- Toggle between device sizes and themes

### Save Behavior
- Auto-save drafts
- Manual publish to push changes live
- Preview unpublished changes

### Version History
- All changes tracked
- Can revert to previous versions
