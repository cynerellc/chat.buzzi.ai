---

## Important Note

**Limited Access**: Company Admins can access operational settings such as escalation rules, webhooks, data collection, and moderation. Core AI settings (model, system prompt, execution environment) are managed by Master Admin and are not visible here.

**Master Admin Only Settings** (configured in Master Admin portal):
- AI Model selection and temperature
- System prompts and core behavior
- Execution environment (memory, timeout, network allowlist)
- Tool configurations and API keys
- Agent package selection
